# 深渊小队 (Abyss Squad)

上帝视角肉鸽小游戏 Mulby 插件。控制 3 人小队在程序生成的地下城中战斗，收集随机能力与道具，触发意想不到的协同组合，通过永久升级变得更强。

## 游戏玩法

### 操作
- **WASD** - 移动活跃英雄
- **空格** - 释放英雄技能（8秒冷却）
- **1/2/3** - 切换控制的英雄
- 其他英雄自动跟随并战斗

### 核心循环
1. 从营地选择 3 人小队出发
2. 在 10 层地下城中战斗
3. 击杀敌人获得经验和水晶
4. 升级时从 3 个随机能力中选 1 个（最多 6 个能力）
5. 拾取随机掉落的道具（武器/神器/饰品 3 个槽位）
6. 特定能力+道具组合触发超强协同效果
7. 每 3 层遭遇 Boss
8. 全灭后回到营地，用水晶进行永久升级

### 英雄（5个）
| 英雄 | 特点 | 技能 |
|------|------|------|
| 战士 | 高血量近战 | 嘲讽：吸引敌人+护盾 |
| 法师 | AOE魔法伤害 | 陨石术：大范围伤害 |
| 游侠 | 远程高暴击 | 闪避射击：后翻+3箭 |
| 牧师（解锁） | 治疗增益 | 治愈光环：全队回血 |
| 刺客（解锁） | 高爆发 | 暗影步：瞬移+必暴击 |

### 协同效果（核心卖点）
集齐特定能力+道具触发超强组合：
- 🔥 **烈焰风暴** = 火焰附魔 + 油瓶 → 全屏火焰伤害
- ⚡ **弹幕风暴** = 多重射击 + 弹射匕首 → 无限弹射
- 🛡️ **血之壁垒** = 吸血攻击 + 荆棘盾 → 攻防一体
- ⚡ **雷神降临** = 连锁闪电 + 导电护符 → 闪电链不限目标
- ⏱️ **时停领域** = 时间减速 + 迅捷之靴 → 敌人静止3秒
- 💥 **核爆** = 分裂弹 + 爆炸宝石 → 分裂弹全部爆炸
- 还有更多组合等你发现...

### 营地（局外升级）
- **训练营**：攻击/生命/移速永久强化
- **铁匠铺**：提升初始武器品质
- **酒馆**：解锁新英雄
- **图书馆**：解锁新能力进入随机池
- **神龛**：解锁新协同组合

## 触发方式
在 Mulby 中输入以下关键词启动：
- `abyss`
- `深渊小队`
- `肉鸽`
