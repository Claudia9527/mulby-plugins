# 热搜排行榜

聚合多平台热搜榜，快捷键驱动，工作之余方便吃瓜。

## 功能

- 聚合 6 大分类、22 个平台的热搜数据
- 全键盘驱动交互，高效浏览
- 插件内阅读模式，一键浏览器打开原文
- 搜索框过滤热搜内容
- 5 分钟自动缓存，减少请求

## 支持平台

| 分类 | 平台 |
|------|------|
| 热搜 | 微博、百度、抖音、今日头条 |
| 社区 | 知乎、V2EX、虎扑、贴吧 |
| 科技 | 36氪、少数派、IT之家、掘金 |
| 视频 | B站、快手、AcFun |
| 资讯 | 澎湃、腾讯、网易、新浪 |
| 生活 | 微信读书、豆瓣、酷安 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `←` / `→` / `Tab` | 切换一级分类 |
| `Ctrl+←` / `Ctrl+→` / `Ctrl+Tab` | 切换二级平台 |
| `↑` / `↓` | 选择列表项 |
| `Enter` | 查看详情 / 返回列表 |
| `ESC` | 关闭阅读页 / 退出插件 |
| `Ctrl+R` | 刷新当前榜单 |
| `Ctrl+O` | 在浏览器中打开原文 |
| `Ctrl+W` | 关闭插件窗口 |

## 触发方式

在 Mulby 搜索框中输入：`热搜`、`hot`、`resou`

## 数据源

使用开源项目 [DailyHotApi](https://github.com/imsyy/DailyHotApi) 作为数据源，默认 API 地址 `https://api-hot.imsyy.top`。
