# 磁盘助手 (Disk Assistant)

一款 Mulby 插件，帮助您分析和可视化磁盘空间占用情况。

## 功能特性

- **目录扫描** - 递归扫描指定目录，统计所有文件和文件夹的大小
- **矩形图可视化** - 使用 Treemap 直观展示空间占用分布，一目了然
- **列表视图** - 支持按大小、名称、类型、日期排序的文件列表
- **搜索过滤** - 按文件名或扩展名搜索，快速定位特定类型的文件
- **磁盘分区信息** - 显示各磁盘分区的总容量、可用空间和已用百分比
- **目录导航** - 面包屑导航，支持点击文件夹深入查看子目录
- **文件定位** - 一键在系统文件管理器中定位文件
- **高性能扫描** - 异步扫描 + 实时进度显示，处理大量文件时不会卡顿

## 使用方式

### 触发命令

在 Mulby 搜索栏输入以下关键词即可打开插件：

- `磁盘` - 打开磁盘助手
- `disk` - 打开磁盘助手
- `空间` - 打开磁盘助手
- `space` - 打开磁盘助手

### 操作流程

1. 打开插件后，点击 **"选择目录"** 按钮选择要分析的目录，或点击磁盘分区按钮扫描整个分区
2. 等待扫描完成，进度条会实时显示扫描状态
3. 扫描完成后：
   - **矩形图视图**：色块大小代表占用空间，点击色块可深入子目录
   - **列表视图**：可排序、搜索的文件列表，支持类型筛选
4. 使用面包屑导航返回上级目录
5. 双击列表中的文件夹可进入查看

## 技术栈

- React 18 + TypeScript
- Tailwind CSS
- Vite
- Lucide React Icons
- Node.js fs API（后端扫描）

## 构建

```bash
# 安装依赖
pnpm install

# 构建
npm run build

# 打包为 .inplugin
npm run pack
```
