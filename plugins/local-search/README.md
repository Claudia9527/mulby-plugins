# 本地搜索

全平台系统文件快速搜索和预览工具。

## 功能

- 调用 Mulby Desktop API 实现跨平台文件搜索
- 三栏布局：左侧文件类型筛选、中间文件列表、右侧文件预览
- 支持多种文件格式预览

### 支持预览的格式

| 类型 | 格式 |
|------|------|
| 图片 | png, jpg, jpeg, bmp, gif, svg, ico, webp |
| 文本 | txt, md, json, xml, yaml, js, ts, py, go 等 |
| 音频 | mp3, wav, flac, m4a, ogg, opus, aac |
| 视频 | mp4, webm, ogv |
| 其他 | 显示文件基本信息 |

### 文件类型分类

全部、图片、表格、文档、音视频、压缩、文本、其他

## 操作指南

| 快捷键 | 操作 |
|--------|------|
| Enter | 打开选中的文件 |
| → (右键) | 显示操作菜单 |
| →→ (连续两次右键) | 直接打开文件所在文件夹 |
| ⌘/Ctrl + C | 复制文件 |
| Tab | 切换文件类型 |
| ↑/↓ | 选择文件 |
| Shift + ↑/↓ | 多选文件 |
| 鼠标拖拽 | 拖拽文件到其他位置 |
| 鼠标右键 | 显示操作菜单 |
| 双击 | 打开文件 |
