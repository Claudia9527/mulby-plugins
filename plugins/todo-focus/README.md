# 待办番茄 (todo-focus)

轻量待办管理插件，支持置顶便签、沉浸式番茄钟，以及 AI 拆解/总结。可通过 Mulby 关键字或划词快速静默添加待办。

## 功能

- **待办列表**：创建、完成、编辑、删除、筛选、置顶焦点任务
- **全键盘操作**：`n` 新建、`j/k` 移动、`Enter` 完成、`e` 编辑、`d` 删除、`/` 搜索
- **便签窗**：独立置顶小窗展示未完成待办
- **番茄钟**：专注 / 短休 / 长休循环，可绑定当前任务
- **AI 助手**：长文本拆解为多条待办；今日待办智能总结
- **快速捕获**：`todo 标题` / `待办 标题` / `td 标题` 静默添加；划词添加

## 触发方式

| 触发 | 说明 |
|------|------|
| `待办` / `todo` / `番茄待办` | 打开主面板 |
| `todo 买牛奶` 等 | 静默添加（正则匹配） |
| 划词 +「添加为待办」 | 静默添加选中文本 |
| `便签` / `待办便签` | 打开置顶便签窗 |
| `专注` / `番茄钟` | 打开番茄钟专注窗 |

## 快捷键（主面板）

| 按键 | 行为 |
|------|------|
| `n` / `Ctrl+N` | 新建 |
| `j` / `k` | 上下选择 |
| `Enter` / `Space` | 完成/切换 |
| `e` | 编辑 |
| `d` / `Shift+D` | 删除 / 直接删除 |
| `p` | 置顶为焦点任务 |
| `/` | 过滤 |
| `f` | 打开专注窗 |
| `s` | 打开便签窗 |
| `1` / `2` / `3` | 全部 / 进行中 / 已完成 |

## 开发

```bash
cd plugins/todo-focus
npm install
npm run build
npm run pack
```

## AI 配置

需在 Mulby 中配置可用 AI 模型；在插件内点击「加载模型」并选择模型后，方可使用拆解与总结功能。

## 手动验收

1. 输入 `待办 测试任务` → 收到通知，主面板可见新待办
2. 划词添加待办
3. 主面板键盘完成新建/完成/删除
4. `便签` 打开置顶窗，单击勾选完成
5. `专注` 开始番茄钟，完成后统计递增
6. AI 拆解与总结（需已配置模型）
7. `npm run pack` 生成 `.inplugin` 可安装
