# mulby-memo (备忘快贴)

一款为 Mulby 设计的剪贴板与片段管理插件，支持快速收藏文本、代码和图片，并能一键快速粘贴。

## 功能特性

- **一键收藏**：支持通过快捷键或关键字，将剪贴板内容（纯文本、代码、图片）默默保存。
- **分类管理**：支持将备忘分为「文本」、「代码」和「图片」三类。
- **快速粘贴**：在界面中点击任意备忘录，即可自动将其粘贴到之前活跃的应用程序中。
- **精美界面**：适配明暗主题，采用 Tailwind 构建的现代化卡片式交互。

## 触发方式

- `备忘快贴` 或 `mulby-memo`：打开插件主界面查看与管理备忘录。
- `默默记录到备忘快贴` 或 `保存为备忘`：不打开主界面，直接在后台将输入框的内容或剪贴板图片保存到数据库。

## 使用方法

1. **收藏文本**：在 Mulby 输入框输入文本后，选择「保存为备忘」。
2. **收藏图片**：在其他应用复制图片后，唤出 Mulby，选择「保存图片为备忘」。
3. **快速调用**：唤出 Mulby 输入「备忘快贴」，在列表中点击需要的片段，即可完成一键自动粘贴。

## 开发

> **💡 提示**: 推荐使用 [pnpm](https://pnpm.io/) 进行依赖管理。

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm run dev
```

### 构建与打包

```bash
pnpm run build
pnpm run pack
```

打包产物为 `.inplugin` 格式，可在 Mulby 插件中心进行离线安装。
